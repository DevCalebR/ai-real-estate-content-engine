# Case Study Blurbs

## 1-Sentence Version

AI Real Estate Content Engine is a Next.js application that turns a short real estate agent brief into a full month of structured social media content, with demo-safe AI orchestration, local persistence, and export-ready deliverables.

## Short Portfolio Version

Built a portfolio-grade AI automation app for real estate marketing that generates a 30-day content calendar, captions, carousel outlines, video scripts, hashtags, and image prompts from a single client brief. The system uses a typed service layer with demo mode and Claude-ready integration, stores generated runs locally, and exports clean markdown, JSON, HTML, and print-friendly reports.

## Job-Application Version

Designed and built an end-to-end AI content automation system in Next.js, TypeScript, and Tailwind that converts structured user input into schema-validated marketing deliverables. The project emphasizes provider abstraction, prompt isolation, formatting pipelines, filesystem persistence, and export workflows rather than a simple prompt UI, making it a strong example of production-minded AI application design.

## Client-Facing Version

I built this system to show how AI can streamline monthly marketing execution without sacrificing structure or polish. A real estate agent can fill out a quick brief and receive a full month of ready-to-use social content, organized inside the app and packaged into clean handoff files for review, design, or publishing.
