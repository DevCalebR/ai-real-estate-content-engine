# Screenshot Checklist

Use these captures for Upwork, a portfolio case study, or LinkedIn project posts.

## Must-Have Screens

- Home dashboard hero with the headline, feature cards, and "Why it feels credible" panel visible
- Generate page with the sample brief loaded and the right-side preview panel visible
- Results page on the Calendar tab showing one or two polished day cards
- Results page on the Carousels tab showing slide-by-slide output
- Results page on the Video Scripts tab showing hook, body beats, and CTA
- Results page on the Exports tab showing all export options
- History page with multiple saved runs visible
- Architecture page showing the workflow cards and plain-language explanation
- Print view / client report page in a clean browser window

## Best Sequence For A Portfolio Carousel

1. Dashboard hero
2. Generate form
3. Results calendar
4. Results exports
5. History page
6. Architecture page
7. Print view / report

## Capture Notes

- Use desktop width for the homepage, results page, and architecture page
- Capture one mobile view for the generate form or results page to show responsiveness
- Prefer screenshots with real sample content rather than empty states
- Keep browser chrome visible on at least one screenshot so it reads as a working application
- Use the included sample brief for consistent screenshots across pages
