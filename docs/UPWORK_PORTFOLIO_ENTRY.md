## Description Options

1. Built Monthly AI Content Engine, a productized web app that turns one structured business brief into a full month of social content. The system supports multiple presets, stores run history locally, and exports clean markdown, JSON, HTML, print-ready reports, and Google Docs deliverables. Demo mode works without AI credentials, while a Claude-ready provider layer supports live generation when configured.

2. Designed and built a niche-agnostic AI content planning application for businesses, agencies, and creators. Users can generate a 30-day content calendar, captions, carousel outlines, short-form video scripts, marketing scripts, hashtags, and image prompts from one workflow, then reopen saved runs and export them into client-ready handoff formats.

3. Shipped a polished Next.js portfolio product that demonstrates real AI automation systems thinking: typed schemas, prompt modules, provider abstraction, local persistence, export workflows, and OAuth-based Google Docs delivery. The app is optimized for reliable demos, including a one-click sample workspace and a demo-safe generation mode.
