# Portfolio Overview

Monthly AI Content Engine is a productized AI content planning application built for businesses, brands, agencies, and creators that need a full month of social content from one structured workflow.

It takes a brief with the business context, offer, audience, goals, tone, platforms, and campaign themes, then turns that into a 30-day content calendar, captions, carousel outlines, short-form video scripts, marketing scripts, hashtags, and image prompts.

What makes the project credible as a portfolio piece:

- it is niche-agnostic instead of hardcoded to one industry
- it includes a demo-safe mode that still exercises the real app workflow
- it stores saved runs locally for repeat demos and exports
- it ships multiple handoff formats, including Google Docs through OAuth
- it exposes the architecture clearly enough for a client or recruiter to understand fast

This is a polished local application, not a fake SaaS shell. It is designed to be easy to demo, easy to explain, and easy to extend later.
